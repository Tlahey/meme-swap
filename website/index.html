Under the hood

Serious AI.
Silly results.

Meme Swap orchestrates FaceFusion, the state-of-the-art open-source face manipulation platform, as a headless engine — wrapped in a fully typed TypeScript pipeline.

Core engine

FaceFusion

The industry-reference face swapping platform, driven headlessly through its Python CLI. Meme Swap installs it in an isolated virtual environment and streams real-time progress back to the UI — analysing, extracting, processing, merging.

Python 3.11 ONNX Runtime CoreML · CPU · CUDA
Face swapper

inswapper_128

InsightFace's flagship swap model (with an fp16 variant for speed) transfers identity frame by frame.

Face enhancer

CodeFormer

Transformer-based face restoration sharpens each swapped face with an adjustable blend.

Frame enhancer

Real-ESRGAN ×2

Optional super-resolution upscales the whole frame for crispier final GIFs.

Expression & lips

Expression Restorer + Lip Syncer

Keeps the original grimaces, blinks and mouth movement so the meme stays the meme.

Media pipeline

FFmpeg

Seamless GIF ↔ MP4 conversion with palette-optimized GIF encoding on the way out.

Built with TypeScript Turborepo + pnpm Next.js 14 Electron Model Context Protocol

Meme Swap for macOS

Native desktop app · Apple Silicon .dmg · guided first-run setup installs FaceFusion for you

Download latest release

The app isn't notarized yet — on first launch, right-click Meme Swap.app → Open.

or build from source
meme-swap — zsh 
# 1. Clone & install the monorepo
$ git clone https://github.com/Tlahey/meme-swap.git && cd meme-swap
$ pnpm install

# 2. One-time: install FaceFusion into ~/.meme-swap
$ pnpm install:facefusion

# 3. Build & launch
$ pnpm build && pnpm frontend:dev
▲ Next.js ready on http://localhost:3010 ✨

Requires macOS with Node.js ≥ 18, pnpm ≥ 9, Python ≥ 3.9 and FFmpeg (brew install ffmpeg). Full guide in the README.
